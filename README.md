# datasciencecoursera
Coursera Signature Data Science project.

Each directory contains materials specific to each course of the Data Science Specialization track offered by John Hopkins. Namely:
- RProgramming
- GettingCleaningData
- ExploratoryAnalysis
- ReproducibleResearch
- StatisticalInference
- RegressionModels
- PracticalML
- DataProduct
- Capstone
